User Guide

The application is divided into independent management sections. Each card below explains how to use its features.

City Management

Create City

  1. Select "Create City" from the Infrastructure dropdown.
  2. Enter the required city and stadium information.
  3. Submit the form to store the city in the database.

View All Cities

  1. Select "View All Cities".
  2. The system will display all stored city records.

Delete City

  1. Select "Delete City".
  2. Enter the city identifier.
  3. Confirm deletion to permanently remove the city.

View / Update / Delete Stadium

  1. Select the desired stadium option from the dropdown.
  2. Enter the stadium identifier if required.
  3. Submit to apply changes.

Team Management

Create Team

  1. Select "Create Team".
  2. Enter team details such as name and group.
  3. Submit to save the team.

View Teams

  1. Select "View Teams".
  2. All stored teams will be displayed.

Update / Delete Team

  1. Select the desired option.
  2. Enter the team identifier.
  3. Submit to apply changes.

Player Management

Create Player

  1. Select "Create Player".
  2. Enter player details such as name, position, and assigned team.
  3. Submit to save the player.

View Players

  1. Select "View Players".
  2. The system will display all player records.

Update / Delete Player

  1. Select the desired option.
  2. Enter the player identifier.
  3. Submit to apply changes.

Game Management

Create Game

  1. Select "Create Game".
  2. Enter match number, teams, date, and stadium information.
  3. Submit to add the game to the database.

View Group Stage Games

  1. Select "View Group Stage Games".
  2. All group stage matches will be displayed.

View Knockout Games

  1. Select "View Knockout Games".
  2. All knockout stage matches will be displayed.

Update / Delete Game

  1. Select the desired option.
  2. Enter the match number.
  3. Submit to apply changes or confirm deletion.
